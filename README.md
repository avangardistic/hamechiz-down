## 🚀 آموزش سریع (2 روش اجرا)

<div dir="rtl">

### روش 1: اجرا با GitHub UI (ساده‌ترین)

۱. ریپازیتوری رو **Fork** کن  
۲. به `Settings` > `Actions` > `General` برو و گزینه `Read and write permissions` رو فعال کن  
۳. به تب `Actions` برو  
۴. سمت چپ، `🍉 همه‌چیزدان v3.0` رو انتخاب کن  
۵. دکمه `Run workflow` رو بزن  
۶. لینک رو در فیلد `manual_url` وارد کن  
۷. کیفیت و تنظیمات دلخواه رو انتخاب کن  
۸. صبر کن تا دانلود تموم بشه  
۹. فایل دانلود شده توی پوشه `downloads/` قرار می‌گیره! 🎉

### روش 2: اجرا با commit message (حرفه‌ای)

```bash
# دانلود ساده
git commit -m "دانلود https://youtu.be/XXXXXX" && git push

# دانلود با کیفیت 1080
git commit -m "دانلود https://youtu.be/XXXXXX quality:1080" && git push

# دانلود کل پلی‌لیست
git commit -m "دانلود https://youtube.com/playlist?list=XXXX playlist:true" && git push

# دانلود فقط صدا
git commit -m "دانلود https://youtu.be/XXXXXX audio-only:true" && git push

# دانلود با زیرنویس فارسی
git commit -m "دانلود https://youtu.be/XXXXXX subtitles:true" && git push
```

## 📝 پارامترهای قابل استفاده در commit message

<div dir="rtl">

| پارامتر | مقادیر | توضیح | مثال |
|---------|--------|-------|------|
| `quality:` | 2160, 1440, 1080, 720, 480, 360, best | کیفیت ویدیو | `quality:1080` |
| `playlist:` | true, false | دانلود کل پلی‌لیست | `playlist:true` |
| `audio-only:` | true, false | فقط صدا (MP3) | `audio-only:true` |
| `subtitles:` | true, false | دانلود زیرنویس | `subtitles:true` |
| `sponsorblock:` | true, false | حذف بخش‌های اسپانسری | `sponsorblock:false` |

### مثال‌های ترکیبی:

```bash
# دانلود 4K با زیرنویس و حذف اسپانسرها
git commit -m "دانلود https://youtu.be/XXXX quality:2160 subtitles:true sponsorblock:true"

# دانلود کل پلی‌لیست با کیفیت 720
git commit -m "دانلود https://youtube.com/playlist?list=XXXX playlist:true quality:720"

# دانلود فقط صدای یک آهنگ
git commit -m "دانلود https://youtu.be/XXXX audio-only:true quality:best"
```

## 🔐 تنظیمات کوکی (برای ویدیوهای خصوصی و سن بالا)

<div dir="rtl">

برای دانلود ویدیوهای زیر به کوکی نیاز دارید:
- ویدیوهای age-restricted (نیاز به تأیید سن)
- ویدیوهای unlisted/private
- ویدیوهای members-only

### آموزش ساخت Secret برای کوکی:

۱. افزونه **"Get cookies.txt LOCALLY"** رو از Chrome Web Store نصب کن  
۲. به YouTube برو و وارد حساب کاربریت شو  
۳. روی افزونه کلیک کن و `Export` رو بزن  
۴. فایل `cookies.txt` دانلود میشه  
۵. محتوای فایل رو کامل کپی کن  
۶. در GitHub برو به `Settings` > `Secrets and variables` > `Actions`  
۷. روی `New repository secret` کلیک کن  
۸. Name: `YOUTUBE_COOKIES`  
۹. Secret: محتوای فایل `cookies.txt` رو paste کن  
۱۰. ذخیره کن  

⚠️ **نکته امنیتی**: از یک حساب ثانویه گوگل استفاده کن و هر 30 روز یکبار کوکی رو به‌روزرسانی کن.

## 🛠️ رفع اشکال

<div dir="rtl">

### ❌ خطا: "هیچ لینکی پیدا نشد"
**راه حل**: 
- مطمئن شو لینک در commit message یا فیلد manual_url وارد شده
- فرمت لینک باید با http:// یا https:// شروع بشه

### ❌ خطا: "Permission denied"
**راه حل**: برو به `Settings` > `Actions` > `General` و دسترسی `Read and write permissions` رو فعال کن

### ❌ دانلود ویدیوهای age-restricted انجام نمیشه
**راه حل**: 
- Secret به نام `YOUTUBE_COOKIES` رو تنظیم کن
- از کوکی جدید استفاده کن (ممکنه منقضی شده باشه)

### ❌ کیفیت 4K دانلود نمیشه
**راه حل**: 
- ممکنه ویدیو کیفیت 4K نداشته باشه
- سعی کن `quality:best` رو امتحان کنی

### ❌ فایل زیپ شده رو چطور اکسترکت کنم؟
**راه حل**: 
- فایل `.zip` رو دانلود کن
- با نرم‌افزارهای معمولی مثل WinRAR یا 7-Zip باز کن
- برای فایل‌های چندبخشی (پارت 001, 002) همه رو در یک پوشه بریز و فایل اول رو اکسترکت کن
