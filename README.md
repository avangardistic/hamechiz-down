# 🍉 همه‌چیزدان

## 📥 دانلودر خودکار از یوتیوب، اینستاگرام، توییتر، تیک‌تاک و هر سایت دیگه‌ای!

## 🚀 آموزش سریع (3 مرحله)

<div dir="rtl">

### مرحله 1: فورک کردن ریپازیتوری

اول این ریپازیتوری رو **Fork** کن (دکمه Fork در بالای صفحه)

### مرحله 2: فعال کردن دسترسی نوشتن

۱. برو به `Settings` ریپازیتوری خودت  
۲. برو به `Actions` > `General`  
۳. توی بخش `Workflow permissions` گزینه `Read and write permissions` رو انتخاب کن  
۴. دکمه `Save` رو بزن  

### مرحله 3: اجرا و دانلود

۱. برو به تب `Actions`  
۲. سمت چپ، `🍉 همه‌چیزدان` رو انتخاب کن  
۳. دکمه `Run workflow` رو بزن  
۴. گزینه `create (ساخت فایل لیست)` رو انتخاب کن و اجرا کن  
۵. فایل `links.txt` که ساخته شده رو ویرایش کن و لینک‌هاتو به این فرمت بنویس:

```txt
https://youtu.be/XXXXXX|video|1080
https://www.instagram.com/p/XXXXXX|video|720
https://twitter.com/XXXX/status/XXXX|video|best
https://soundcloud.com/XXXX/XXXX|audio|best
```

۶. دوباره برو به `Actions` و `Run workflow` رو بزن  
۷. این بار گزینه `download (دانلود از روی لیست)` رو انتخاب کن  
۸. صبر کن تا دانلود تموم بشه  
۹. فایل‌های دانلود شده توی ریپازیتوری تو قرار می‌گیرن! 🎉

</div>

## 📝 فرمت فایل لیست (links.txt)

<div dir="rtl">

فایل `links.txt` باید به این شکل باشه:

```
لینک|نوع|کیفیت
```

| قسمت | توضیح | مقادیر مجاز |
|------|-------|-------------|
| **لینک** | آدرس کامل ویدیو/صدا | هر لینک معتبر |
| **نوع** | نوع دانلود | `video` یا `audio` |
| **کیفیت** | کیفیت مورد نظر | `2160`, `1440`, `1080`, `720`, `480`, `360`, `best` |

### مثال‌ها:

```txt
# دانلود ویدیو با کیفیت 1080 از یوتیوب
https://www.youtube.com/watch?v=dQw4w9WgXcQ|video|1080

# دانلود فقط صدای یک اهنگ از یوتیوب
https://youtu.be/XXXXXXXXXXX|audio|best

# دانلود ریل اینستاگرام
https://www.instagram.com/reel/XXXXXXXXX|video|720

# دانلود ویدیو از توییتر
https://twitter.com/XXXX/status/XXXXX|video|best

# دانلود موزیک از ساوندکلاود
https://soundcloud.com/XXXXX/XXXXX|audio|best
```

</div>

## ❗ نکات مهم

<div dir="rtl">

1. **فضای ذخیره‌سازی**: ریپازیتوری رایگان گیت‌هاب حدود 4GB فضا داره

## 🛠️ رفع اشکال

<div dir="rtl">

### ❌ خطا: "فایل links.txt پیدا نشد"
**راه حل**: اول گزینه `create (ساخت فایل لیست)` رو اجرا کن تا فایل ساخته بشه

### ❌ خطا: "Permission denied"
**راه حل**: برو به `Settings` > `Actions` > `General` و دسترسی `Read and write permissions` رو فعال کن

### ❌ دانلود بعضی لینکها دچار خطامی‌شه
**راه حل**: 
- مطمئن شو لینک درست و عمومی هست
- سعی کن کیفیت رو `best` یا `720` بذاری
- برای اینستاگرام از لینک پست عمومی استفاده کن

### ❌ فایل زیپ شده رو چطور دانلود کنم؟
**راه حل**: توی ریپازیتوری، روی فایل `.zip` کلیک کن و `Download` رو بزن
